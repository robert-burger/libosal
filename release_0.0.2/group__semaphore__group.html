Detailed Description

The semaphores are a synchronization mechanism for 2 or more task with a counter value.

Initialize a semaphore.

This function initializes a semaphore structure given by sem. If passed attr is NULL a default semaphore is initialized. The counter is initilized to initval.

Parameters
[in]semPointer to osal semaphore structure. Content is OS dependent.
[in]attrPointer to initial semaphore attributes. Can be NULL then the defaults of the underlying mutex will be used.
[in]initvalInitial semaphore cound value
Return values
OSAL_OKOn success.
OSAL_ERR_NOT_IMPLEMENTEDShared was requested but there's not support from OS.
OSAL_ERR_INVALID_PARAMInvalid input parameter.

◆ osal_semaphore_post()

osal_retval_t osal_semaphore_post ( osal_semaphore_t *  sem)

Post a semaphore.

This function "posts" the semaphore. That means it increments the internal counter value and unblocks waiting tasks on that semaphores to continue their execution.

Parameters
[in]semPointer to osal semaphore structure. Content is OS dependent.
Return values
OSAL_OKOn success.
OSAL_ERR_INVALID_PARAMInvalid input parameter.
OSAL_ERR_OPERATION_FAILEDError occuered posting semaphore e.g. counter is at maximum value.

osal_retval_t osal_semaphore_timedwait ( osal_semaphore_t *  sem,
const osal_timer_t *  to 
)

Wait for a semaphore.

Wait for a semaphore to become available. If the internal counter is already greater than 0 it decrements the counter and return OSAL_OK immediately. If the counter is 0 it will block until some other tasks call osal_semaphore_post or a timeout or any error occures.

Parameters
[in]semPointer to osal semaphore structure. Content is OS dependent.
[in]toTimeout.
Return values
OSAL_OKOn success.
OSAL_ERR_INTERRUPTEDCall was interrupted by a signal during wait.
OSAL_ERR_INVALID_PARAMInvalid input parameter.
OSAL_ERR_TIMEOUTTimeout occured waiting for semaphore to become available.

osal_retval_t osal_semaphore_trywait ( osal_semaphore_t *  sem)

Try to wait for a semaphore but don't block.

If the semaphore counter is greater than 0 it decrements the counter and returns OSAL_OK immediately. If the counter is 0 it returns OSAL_ERR_BUSY and does not block!

Parameters
[in]semPointer to osal semaphore structure. Content is OS dependent.
Return values
OSAL_OKOn success.
OSAL_ERR_BUSYWaiting for semaphore would block.
OSAL_ERR_OPERATION_FAILEDOther error occuered.
