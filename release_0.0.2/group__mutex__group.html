Detailed Description

The mutexes are mutual exclusion locks which are commonly used to protect shared memory structures from concurrent access.

◆ osal_mutex_init()

osal_retval_t osal_mutex_init ( osal_mutex_t *  mtx,
const osal_mutex_attr_t *  attr 
)

Initialize a mutex.

This function initializes a mutex structure given by mtx. If no attributes are given with attr a default mutex is initiazed.

Parameters
[in]mtxPointer to osal mutex structure. Content is OS dependent.
[in]attrPointer to initial mutex attributes. Can be NULL then the defaults of the underlying mutex will be used.
Return values
OSAL_OKOn success.
OSAL_ERR_SYSTEM_LIMIT_REACHEDNot enough system resources.
OSAL_ERR_OUT_OF_MEMORYSystem is out of memory.
OSAL_ERR_PERMISSION_DENIEDPermission denied opening a shared mutex.
OSAL_ERR_INVALID_PARAMInvalid input parameter.
OSAL_ERR_UNAVAILABLEOther errors.
