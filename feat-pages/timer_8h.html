Detailed Description

OSAL timer header.

Author
Robert Burger rober.nosp@m.t.bu.nosp@m.rger@.nosp@m.dlr..nosp@m.de
Date
07 Aug 2022

This functions implements timers for use in realtime systems. They can be used to implement timeouts, do a deterministic sleep for some amount of time or just to get the current system time.
