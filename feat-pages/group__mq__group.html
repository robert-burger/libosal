Detailed Description

Message queues are an asynchronous communication mechanism between two or more processes/tasks. They follow the publish/subscribe pattern.

Initialize a mq.

Parameters
[in]mqPointer to osal mq structure. Content is OS dependent.
[in]namePointer containing message queue name.
[in]attrPointer to initial mq attributes. Can be NULL then the defaults of the underlying mq will be used.
Return values
OSAL_OKOn success.
OSAL_ERR_PERMISSION_DENIEDTried to open an existing queue without sufficient permission. Tried to create an already existing message queue.
OSAL_ERR_INVALID_PARAMName is not in correct format or it is too long.
OSAL_ERR_SYSTEM_LIMIT_REACHEDLimit of open files has been reached.
OSAL_ERR_NOT_FOUNDTried to open a non-existing message queue.
OSAL_ERR_OUT_OF_MEMORYSystem is out of memory.
OSAL_ERR_OPERATION_FAILEDOther errors.
