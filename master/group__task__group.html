const osal_task_attr_t *  attr,
osal_task_handler_t  handler,
osal_task_handler_arg_t  arg 
)

Create a task.

Parameters
[in]hdlPointer to osal task structure. Content is OS dependent.
[in]attrPointer to initial task attributes. Can be NULL then the defaults of the underlying task will be used.
[in]handlerTask handler to be executed.
[in]argPointer to argument passed to task handler.
Return values
OSAL_OKOn success.
OSAL_ERR_SYSTEM_LIMIT_REACHEDSystem is out of resources.
OSAL_ERR_PERMISSION_DENIEDPermission denied for priority/policy.
OSAL_ERR_INVALID_PARAMInvalid input parameter.
OSAL_ERR_OPERATION_FAILEDOther errors.
